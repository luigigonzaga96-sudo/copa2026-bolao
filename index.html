❓ Dúvidas Frequentes
Posso palpitar depois do jogo começar? +
Não. O sistema bloqueia automaticamente 5 minutos antes do kickoff. Faça seu palpite com antecedência!
Posso mudar meu palpite depois de salvar? +
Sim! Quantas vezes quiser, até 5 min antes do jogo. O último palpite salvo é o que vale.
Posso usar IA para escolher meu palpite? +
Pode! Mas IA não prevê o futuro — só analisa probabilidades históricas. Faz parte da estratégia 😄
Usei celular e PC — qual palpite vale? +
O último salvo, em qualquer dispositivo. Não é possível ter dois palpites diferentes por jogo.
Qual o horário para palpitar? +
A janela de palpites é das 05:00h às 12:30h todos os dias. Fora desse horário os palpites ficam bloqueados.

💡 Dica de ouro: antecipe já os palpites da próxima rodada inteira assim que a janela abrir — você pode alterar à vontade até às 12:30h. Assim nunca perde um jogo por falta de tempo!
Posso alterar meu palpite depois de salvar? +
Sim! Quantas vezes quiser, desde que a janela esteja aberta (05:00h–12:30h). O último palpite salvo dentro da janela é o que vale.
Como funciona a pontuação? +
5 pontos — Placar exato 🎯

3 pontos — Resultado certo (V/E/D) mas errou o placar

0 pontos — Errou
Como funciona o Ranking Anytools? +
Compara unidades pela média de pontos por participante. Unidade com menos pessoas não leva desvantagem.
Como os resultados são atualizados? +
Automaticamente via API: a cada 5 minutos durante jogos e a cada 10 minutos fora. O admin pode corrigir manualmente no painel.
Quem pode participar? +
Acesso exclusivo por convite. Somente e-mails cadastrados pelo admin conseguem criar conta.
Esqueci minha senha. O que faço? +
Na tela de login, clique em "Esqueci minha senha". Um link de recuperação é enviado ao seu e-mail automaticamente — sem acionar o admin.
🔎 As ferramentas do projeto
GitHub
Onde o código fica guardado e publicado. Segurança: chaves privadas não devem estar aqui.
Firebase
Login e banco de dados (Google). Segurança: regras do Firestore controlam quem acessa o quê.
API-Sports
Placares em tempo real. Segurança: chave no código — risco de cota, não de dados dos usuários.
Meus dados estão seguros? +
Sim. Dados ficam no Firebase (Google). Só você vê seus palpites antes do jogo. Nenhum dado financeiro é coletado.
Alguém de fora pode ver os dados? +
Não. Acesso requer login válido. As regras do Firestore bloqueiam qualquer leitura sem autenticação.
O admin pode ver minha senha? +
Não. Senhas são gerenciadas 100% pelo Firebase (Google) com criptografia irreversível. Ninguém tem acesso.
Preciso me preocupar com LGPD? +
Para uso interno entre colaboradores, a base legal é sólida. Os dados (nome e e-mail) são usados só para o bolão e removidos após a Copa 2026.
